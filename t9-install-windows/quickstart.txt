1. 제품 T9 을 실행하고 종료하는 방법입니다.

 1) 실행하기
	cd %T9_HOME%/bin
	./run.sh	
	
 2) 종료하기
	cd %T9_HOME%/bin
	./stop.sh

 3) 로그 확인 
	tail -f %T9_HOME%/logs/rosemary.log
